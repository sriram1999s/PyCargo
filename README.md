# PyCargo
## A Package Manager for Python, similar to Rust's Cargo

## Functionalities

  - [new](#Creating-Package)
  - [run](#Running)

# Installation
```python
print("Yet to complete :D")
```
# Usage

## Creating Package

creating a new package in the current working directory

```shell
PyCargo new <name>
```

## Running

make sure to be in a PyCargo managed directory!

```shell
PyCargo run
```
